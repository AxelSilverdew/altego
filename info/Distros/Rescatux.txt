**Rescatux** is a Debian-based GNU/Linux live distribution that includes a graphical wizard for rescuing broken GNU/Linux installations.
**Notable features**
- Create UEFI Boot entries.
- Fake Microsoft Windows UEFI paths.
- Restore GRUB and GRUB2.
- Update any GRUB2 boot menues.
- Restore Windows MBR.
- Clear Windows passwords.
- File System Check.
- Gparted.
- and many more.
**Available DEs**
- Openbox
